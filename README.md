# Cyber Threat Intelligence Dashboard

## Overview

A Cyber Threat Intelligence Dashboard developed using Python and Streamlit that performs real-time IP reputation analysis using the AbuseIPDB API.

## Features

- Real-time IP reputation analysis
- Threat score calculation
- Automated risk classification
- Country, ISP, Domain and TOR node detection
- Interactive dashboard
- Downloadable threat reports

## Technologies Used

- Python
- Streamlit
- Plotly
- AbuseIPDB API
- REST API
- JSON
